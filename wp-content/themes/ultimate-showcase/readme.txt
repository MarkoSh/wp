== Description ==
The Ultimate Showcase theme provides a sleek, modern and responsive layout that allows you to display your best work in a clear and attractive manner. This theme is designed, from the ground up, to integrate seamlessly with the <a href="https://wordpress.org/plugins/ultimate-product-catalogue/">Ultimate Product Catalog</a> plugin. It's easy to set up and offers you the perfect venue to showcase your products and/or portfolio.

* Modern responsive layout
* Ultimate Product Catalog integration
* Feature-rich slider that pulls slides either from a dedicated custom post type, straight from your product catalog or from WooCommerce
* Testimonials
* Multiple page layouts
* Blog layout
* Theme options
* Flexible homepage layout

For more information about the Ultimate Showcase theme, you can visit <a href="http://etoilewebdesign.com/ultimate-showcase/">http://etoilewebdesign.com/ultimate-showcase/</a>.

== Installation ==

1. In your WordPress admin panel, go to Appearance > Themes and click the "Add New" button.
2. Type "Ultimate Showcase" in the search box and press Enter.
3. Click the "Activate" button to use the theme right away.
4. Go to Theme Options (in the sidebar in your WordPress admin) to customize the theme to your needs.

OR

1. Upload the "ultimate-showcase" folder to the "/wp-content/themes/" directory
2. Activate the theme through the Appearance > Themes menu in WordPress
3. Go to Theme Options (in the sidebar in your WordPress admin) to customize the theme to your needs.
